# Gasless self-funding — research (2026-05-31)

Goal: the tool funds its own gas with no human — convert a little of our stablecoin into POL
without already holding POL.

## Real solutions found (use one)
- 0x Gasless API (api.0x.org/gasless/quote, chainId 137, header 0x-version: v2): swaps an ERC-20 gaslessly;
  gas is recouped from the SOLD token. Supports Polygon + USDC. Min ~$1 on Polygon. Same fee params we already
  use (swapFeeRecipient/swapFeeBps/swapFeeToken). LIMIT: cannot SELL the native coin gaslessly (we BUY POL, fine).
  Source: 0x.org/docs/gasless-api
- Polygon native "Swap For Gas": converts bridged stablecoins -> POL gaslessly, min 1 / max 20 POL.
  Source: polygon.technology/blog/swap-for-gas
- MetaMask / Coinbase gasless swaps (smart-account based) do the same; reference only.

## Our holdings (live 2026-05-31)
- proxy 0x0a94: 70.90 pUSD + 6.0 USDC.e + 0 POL.
- signer EOA 0x646 (we hold the key): 38.72 pUSD + 0 USDC.e + 0 POL.

## The wrinkle to solve
- 0x gasless needs a STANDARD token (USDC.e) held by a signable EOA (the taker signs a permit/meta-tx).
- Our standard token (6 USDC.e) is in the PROXY contract, not a plain EOA; our EOA holds pUSD (Polymarket's
  token, likely no gasless/DEX liquidity). So the path is: get USDC.e into the signer EOA (via Polymarket's
  gasless rails or a permit), then 0x-gasless swap USDC.e -> POL. Confirm pUSD<->USDC.e route at build.
